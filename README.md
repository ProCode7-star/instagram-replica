# 📸 Réplica de Instagram

¡Hola! Este es un proyecto que replica la interfaz móvil tradicional de **Instagram** utilizando tecnologías web esenciales. Está diseñado con un enfoque responsivo, imitando la experiencia clásica de la aplicación móvil directamente en el navegador.

---

## 🚀 Características
*   **Barra superior:** Logotipo clásico y botones de interacción rápida (Notificaciones y Mensajes).
*   **Sección de Historias:** Carrusel horizontal con el clásico borde degradado de Instagram.
*   **Feed de Publicaciones:** Tarjeta de post con diseño limpio, sección de interacciones, contador de likes y pie de foto.
*   **Menú Inferior:** Barra de navegación fija para simular el cambio entre pestañas.
*   **Diseño Claro (Light Mode):** Estética limpia y fiel a la interfaz original.

---

## 🛠️ Tecnologías Utilizadas
*   **HTML5:** Estructura semántica del clon.
*   **CSS3:** Estilos personalizados, layouts con **Flexbox** y variables de diseño.
*   **FontAwesome:** Iconos vectoriales para los botones de la interfaz.
*   ¡Y más! (Próximamente integrando scripts interactivos y automatizaciones).
